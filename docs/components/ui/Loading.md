# Loading

### About

Wrapper for antd loading component. Main purpose is to standardise the loading animation

### Parameters

```javascript
const propTypes = {
  className: PropTypes.string,
};

const defaultProps = {
  className: 'loading-spinner',
};
```

### Example

```javascript
import { Loading } from '@spotlightdata/nanowire-extensions';

render(<Loading />);
```
